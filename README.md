# WT-II-Project

Online learning portal:

-Login/Registration Page

-Home Page : (Before logging in)
    List of courses With categories
    Testimonials
    Search Box for courses

-Home Page (After login):
    List of enrolled courses
    Suggested courses

-Course Page:
    About the course
    Instructors
    Duration
    FAQ
    Enrollment Link
    Payment Portal 
    
-Profiles of teachers
    

Front end
Vue.js
React

Backend
Python Flask/Django
Node JS
Mongo
